# 「ホロ・グラム」- ホロライブVtuberの配信予定確認アプリ -

## 概要
ホロライブVtuberの配信を確認するためのアプリです。
本アプリの特徴は主に以下の3点です。
1. 誰が何時に配信するのか分かりやすい配信予定画面
2. ソート、フィルター機能
3. カテゴリ、出演者を表示したタグ機能

※当アプリはファンが個人で制作している非公式のものであり、公式とは一切関係ございません。
